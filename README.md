# pan26-ai-service

AI-powered backend service for academic plagiarism detection — semantic search and document retrieval API.

## Overview

This service is the AI backbone of a generative plagiarism detection system built for PAN 2026. Given a suspicious document believed to be generated from one or more academic source papers, the service retrieves the most likely source documents from a corpus of sources.

It is designed as a standalone, API-ready module that can be integrated into a larger software system.
